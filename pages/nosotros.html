Sobre Nosotros

¿Quiénes Somos?

Somos estudiantes apasionados por el desarrollo web, creando proyectos educativos que combinan teoría y práctica para aprender tecnologías modernas de frontend.

Nuestra Misión

Proporcionar herramientas prácticas y ejemplos reales para que otros estudiantes puedan aprender desarrollo web de manera efectiva y divertida.

El Proyecto

Este sitio es parte de nuestro aprendizaje en desarrollo frontend. Incluye páginas estáticas, formularios interactivos, gestión de tareas con localStorage y diseño responsive.

Tecnologías Usadas

  • HTML5
  • CSS3 con Bootstrap
  • JavaScript (ES6+)
  • localStorage API

Características del Sitio

Administrador de Tareas

Crea, edita y organiza tus tareas con filtros y búsqueda.

Diseño Responsive

Funciona perfectamente en desktop, tablet y móvil.
